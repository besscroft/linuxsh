## linuxsh

自用的 sh 脚本

### centos-amd64 适用于 centos7 x86_64 机器

```shell
wget -N --no-check-certificate "https://raw.githubusercontent.com/besscroft/linuxsh/master/centos-amd64.sh"
chmod +x centos-amd64.sh
./centos-amd64.sh
```
